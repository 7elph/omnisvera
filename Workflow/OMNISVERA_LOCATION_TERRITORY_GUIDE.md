# Omnisvera — Guia de Territórios e Locais

Fonte da verdade relacionada: [[OMNISVERA_SYSTEM_TAXONOMY]]

## Regra simples

| Campo | Uso |
|---|---|
| `territory` | área grande: reino, continente, região marítima, floresta regional |
| `location` | ponto jogável: cidade, vila, fortaleza, santuário, ruína, estrada, lago |
| `region` | subdivisão opcional dentro de um território |
| `district` | distrito/bairro/setor dentro de uma cidade ou capital |

## Exemplos

| Coisa | Classificação |
|---|---|
| Earthropo | território / continente |
| Reino de Nimalia | território / reino |
| Nimalis | local / capital |
| Floresta de Avenor | território ou região florestal |
| Bosque Sussurrante | local / floresta |
| Vale Dourado | local / vila remota ou região agrícola menor |
| Fortaleza de Gharok | local / fortaleza |
| Mar da Neblina | território / região marítima |

## Regra de uso

- Se serve como cenário amplo de várias aventuras, usar `territory`.
- Se é um lugar onde uma cena acontece, usar `location`.
- Se é uma subdivisão de território, usar `region`.
- Se é parte de cidade, usar `district`.
