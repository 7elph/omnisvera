## Regra para mídia

Nenhuma mídia deve ser apagada ou movida até que:

1. referências Markdown sejam verificadas;
2. frontmatter seja verificado;
3. Leaflet seja verificado;
4. Calendarium seja verificado;
5. CSS/snippets sejam verificados;
6. JSON de plugins seja verificado;
7. o Sage aprove a remoção ou arquivamento.
