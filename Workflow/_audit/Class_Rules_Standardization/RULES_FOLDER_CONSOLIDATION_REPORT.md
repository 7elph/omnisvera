# Relatório — Consolidação de Classes e Raças

## Decisão Aplicada

`Classes/` e `Races/` são as pastas oficiais para classes e raças.

As pastas `Rules/Classes` e `Rules/Races` foram esvaziadas e removidas porque duplicavam uma estrutura que já existia no vault.
